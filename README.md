# ProjectTBA
Initial Project for an Upcoming Pirate Game, placeholder name Tides of the Broken Archipelago. 
Game Made using GODOT. 
Future GOTY 2028.
A 4 player coop arcade roguelite pirate game. 
With Chatioc gameplay, with rigidbody boat physics, and physics modifiers to change the rules of the game at every new RUN.
